# 03: Fundação de Dados — Tabela `mentee_briefing` e Token por Mentorado

**Tipo:** Implementação
**Página:** Módulo A / Módulo B (base para tudo)

## Descrição
Criar a estrutura de dados que sustenta a feature: tabela `mentee_briefing` com um registro por mentorado e um campo por pergunta das 29 perguntas, mais campos de token único, status (pendente/preenchido), marcação de "revisão pendente" e timestamps (criado, preenchido, atualizado). Inclui a definição compartilhada das seções e campos das 29 perguntas reutilizada pelo formulário e pela ficha.
