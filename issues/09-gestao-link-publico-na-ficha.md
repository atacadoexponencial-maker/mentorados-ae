# 09: Frontend — Gestão do Link Público na Ficha

**Tipo:** Implementação
**Página:** Módulo B — Visualização e Gestão do Briefing na Ficha Interna

## Descrição
Implementar, dentro da seção de briefing do drawer, os controles de link público: botão de gerar link quando ainda não existe, exibição do link existente, botão de copiar para a área de transferência e botão de revogar/regenerar (consumindo os endpoints de token).
